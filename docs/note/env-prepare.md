# 环境准备

## Docsify
用于快速搭建简约文档系统，官网地址 [https://docsify.js.org/](https://docsify.js.org/)

```bash
npm i docsify-cli -g
```

```bash
docsify init ./docs
```

```bash
docsify serve ./docs --port 3001
```

## Hexo

```bash
npm install -g hexo-cli
```

```
$ hexo init <folder>
```
